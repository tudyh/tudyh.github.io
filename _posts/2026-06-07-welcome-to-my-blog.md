This is the first post on my blog. To write a new one, add a file to `_posts/`
named `YYYY-MM-DD-title-of-post.md` with the same front matter as this one
(`layout: post`, `title`, `date`, `description`), then write the post content
in Markdown below the front matter. It will automatically show up on the
[blog page](/blog/), most recent first.
